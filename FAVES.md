# Favourites page

## Sections

- Movies
- TV shows
- Music (albums)

## Layout

- Omit the normal website header and theme switcher
- Movies should be grouped into genres with 4 films in each genre
- The 4 movies should be displayed in a row, with the text underneath them. Be creative with space efficiency.
- TV shows should display in a grid; fit as many as possible
- The design should be responsive and work on mobile
- There should be at least two modes -- one with all information displayed, and one with only covers. If you have other ideas for useful modes, add them.
- You should be able to sort by release date
- The design should be simple and functional; refer to Letterboxd as an example, but don't copy it entirely
